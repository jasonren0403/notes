## 考试形式和试卷结构
1. 试卷满分及考试时间
   本试卷满分150分，考试时间为180分钟。
2. 答题方式
   答题方式为闭卷、笔试。
3. 试卷内容结构

    ```mermaid
    pie
        "数据结构(45分)" : 45
        "计算机组成原理(45分)" : 45
        "操作系统(35分)" : 35
        "计算机网络(25分)": 25
    ```

4. 试卷题型结构

    ```mermaid
    pie
    "单项选择题(80分)" : 80
    "综合应用题(70分)" : 70
    ```

	* 选择题40个，每题2分
